 Task Management Application

A full-stack Task Management Application built with Laravel 10 (REST API) and Vue.js 3 frontend.

 Tech Stack
Layer	Technology
Backend	Laravel 10+, PHP 8.1+
Frontend	Vue.js 3
Database	MySQL 
Testing	PHPUnit
 Features

Create, Read, Update, Delete tasks

Task status management

RESTful API

Validation using Form Requests

API responses using Resources

Feature tests

Database seeding

 Task Fields
Field	Type	Required
title	string (max 255)	
description	text	
status	enum (pending, in_progress, completed)	
due_date	date
